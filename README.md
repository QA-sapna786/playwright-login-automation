# Playwright Login Automation Framework
This project is an automation framework built using Playwright to test the login functionality of a web application.

## Project Overview
- Automates login functionality of SauceDemo website
- Uses Page Object Model (POM) for better structure
- Implements reusable and scalable test design

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Project Structure
tests/
  ├── login.spec.js
  ├── loginpage.js
  ├── logincredentials.js

## Features
✔ Page Object Model (POM)  
✔ Clean and reusable code  
✔ Easy to maintain  
✔ Scalable for large projects  

## How to Run
1. Install dependencies:
```bash
npm install
2. Execute tests:
npx playwright test
